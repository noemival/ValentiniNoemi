TODO (caso di studio ledAsAlarm): turn on a led when a sonar detected a distance d<=V. V is an integer value preixed, it is between 5 and 80.

Requirement analysis

  • led: a light-emitting diode (a semiconductor diode which glows when a voltage is applied).
  • distance: a distance between the sonar and everething element is detected by the sonar.

    • Users Story

    The user just have to launch the application and put something at the correct distance to verify if the led turn on

    Problem analysis

    The application is based on the interaction of two components which are a sonar and a led. It is not specify where this components are situated. So the most important specifies to considerate are:
  • Comunication between the components: The sonar should be listening if something will be near itself, then it should be able to send a command to the led and make it on. QActors distribuited system, are useful to implement this application because they could helped for the comunication and make an easly implementation.
  • Tecnology indipendence
    Both this problems was analyzed in ButtonLed_es01
